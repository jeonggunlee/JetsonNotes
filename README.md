# JetsonNotes

## jtop install
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install python3-pip
sudo -H pip3 install -U jetson-stats

sudo reboot
jtop
```

## onnx install

```
sudo python3 -m pip install onnx
```
